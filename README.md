# 微博助手·历史清理·媒体停播

一个基于 Chrome Manifest V3 的轻量扩展，包含三项核心能力：

- 微博发帖文本半自动填充
- 黑名单域名历史记录自动清理
- 媒体倒计时自动停止（到点暂停所有标签页音视频）

## 功能说明

### 1. 历史记录自动清理

- 支持在弹窗中维护域名黑名单（增删改查）
- 切换标签页时，若命中黑名单域名，自动清理对应历史记录
- 支持“立即清理”按钮，手动触发一次全量清理
- 若后台短暂未连接，弹窗会自动重试请求

### 2. 微博发帖辅助

- 仅在 `https://weibo.com/*` 注入内容脚本
- 监听微博编辑按钮点击，读取最新打卡文本并生成下一条内容
- 自动填充到发帖输入框（`textarea`）

### 3. 媒体自动停止

- 在弹窗输入倒计时分钟数（1-1440）后可启动倒计时
- 倒计时结束后自动暂停所有标签页中的 `video/audio`
- 可手动停止倒计时
- 倒计时结束支持通知提醒（默认开启）

## 安装方式（开发者模式）

1. 打开 `chrome://extensions`
2. 打开右上角“开发者模式”
3. 点击“加载已解压的扩展程序”
4. 选择当前项目根目录
5. 每次代码改动后点击扩展卡片上的 `Reload`

## 使用说明

### 历史清理配置页

1. 点击浏览器工具栏扩展图标，打开弹窗
2. 在输入框中添加域名（如 `weibo.com`）
3. 可在列表中删除域名
4. 点击“立即清理”可立刻执行一次历史清理

### 微博辅助

1. 打开微博页面并进入发帖/编辑流程
2. 点击触发按钮后，扩展会尝试自动生成并填充文本

### 媒体自动停止

1. 点击扩展图标，进入“媒体自动停止”区域
2. 输入分钟数并点击“开始倒计时”
3. 状态会实时显示剩余时间
4. 需要提前结束时点击“停止倒计时”

## 项目结构

- `manifest.json`：扩展声明与权限配置
- `js/background.js`：后台服务（黑名单同步、自动/手动历史清理）
- `js/popup.js`：弹窗逻辑（黑名单管理、立即清理、媒体倒计时控制）
- `popup.html`：弹窗结构
- `popup.css`：弹窗样式
- `js/content.js`：微博页面内容脚本（纯原生 JS，无 jQuery）
- `img/icon-*.png`：扩展图标资源

## 权限说明

- `tabs`：读取当前激活标签页 URL，用于判断是否命中黑名单
- `storage`：保存黑名单配置（`chrome.storage.sync`）
- `history`：搜索并删除命中域名的浏览历史
- `scripting`：向标签页注入暂停媒体脚本
- `alarms`：管理媒体停止倒计时
- `notifications`：倒计时结束通知
